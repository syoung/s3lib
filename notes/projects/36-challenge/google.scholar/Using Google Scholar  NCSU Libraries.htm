Using Google Scholar with NCSU Libraries

Tips!
  • More tips on using Google Scholar

To use Google Scholar effectively, you want to make sure you get access to the full text of articles via NCSU Libraries.

 

Use any of the 3 methods below:

 

Tell Google Scholar you are affiliated with NCSU


1. If you have a Google/Gmail/Blogger account, log in first. This will ensure that the preferences you set in Step 2 will stick.
2. Go to Google Scholar preferences.
3. In "Library Links," add "North Carolina State University (FullText@NCSU) "

3. "Save Preferences."

 

Bookmark this page and use this search box:

 

Bookmark this URL:   http://www.lib.ncsu.edu/cgi-bin/proxy.pl?server=scholar.google.com

If you have any trouble getting the articles or books you want, Ask Us
