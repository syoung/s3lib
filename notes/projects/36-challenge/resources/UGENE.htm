[edit] Features

UGENE has modular structure and almost all its functionality comes within plugins. Internally UGENE is designed with support of parallelism to make profit from using multicore computers. Every computational task in UGENE is performed in background and does not block user interaction with visual interface.

  • Visual representation of annotated DNA or protein sequences
  • Huge files support, native support of gzipped documents
  • Chromatograms visualization
  • Search for ORFs
  • Integrated support of search requests to NCBI databases
  • Multiple sequence alignment with port of MUSCLE
  • Easy-to-use search with HMM profiles using adopted HMMER package
  • Integrated 3D viewer for files in PDB format
  • Designer for custom computational workflows
  • Search for transcription factor binding sites TFBS in DNA sequences
  • Restriction enzymes search and integration with REBASE restrictions enzymes database
  • Search for direct and inverted repeats in DNA sequences
  • A complete Smith-Waterman algorithm implementation
